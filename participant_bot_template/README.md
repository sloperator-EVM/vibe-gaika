# Пример бота для GAICA

Этот шаблон снова ориентирован на **явную боевую логику**, а не на AI/веса модели.

Что внутри:

- `main.py` — точка входа, запускает `CombatBot`
- `gaica_bot/models.py` — OOP-модели сообщений и состояния игры
- `gaica_bot/client.py` — сетевой клиент и цикл обработки сообщений
- `gaica_bot/combat_bot.py` — основной боевой бот с логикой оружия / линии огня / уклонения
- `gaica_bot/sample_bot.py` — старый минимальный пример
- `tests/test_combat_bot.py` — проверки ключевых боевых решений

## Логика `CombatBot`

Бот делает именно такой цикл:

1. идёт к ближайшему доступному оружию и подбирает его;
2. после подбора пытается выйти на чистую линию огня по противнику;
3. если линия чистая — держит боевую дистанцию и стреляет;
4. если в него летит опасный снаряд — уходит в сторону от траектории;
5. если враг слишком близко — бьёт ногой.

## Быстрый старт

```bash
python participant_bot_template/main.py 127.0.0.1 9001
```

Запуск браузерного матча из корня репозитория:

```bash
python scripts/start_browser_match.py \
  --bot-a-cmd "{python} participant_bot_template/main.py {host} {port}" \
  --bot-b-cmd "{python} participant_bot_template/main.py {host} {port}"
```
