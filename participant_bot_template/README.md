# Пример бота для GAICA

Этот шаблон ориентирован на **явную боевую логику**, а не на AI/веса модели.

Что внутри:

- `main.py` — точка входа, запускает `CombatBot`
- `gaica_bot/models.py` — OOP-модели сообщений и состояния игры
- `gaica_bot/client.py` — сетевой клиент и цикл обработки сообщений
- `gaica_bot/combat_bot.py` — основной боевой бот с приоритетами: подобрать оружие, не вылететь с карты, занять линию атаки, пережить снаряды
- `gaica_bot/sample_bot.py` — старый минимальный пример
- `tests/test_combat_bot.py` — проверки ключевых боевых решений

## Логика `CombatBot`

Бот делает именно такой цикл:

1. если оружие лежит рядом — подбирает его сразу;
2. если оружия нет — идёт к лучшему доступному pickup через pathfinding;
3. если есть опасный снаряд — уклоняется только в допустимое walkable-направление;
4. если линия прострела перекрыта ломаемым объектом — ломает его;
5. если враг вооружён и подошёл слишком близко — пытается выбить оружие ногой;
6. иначе занимает выгодную боевую позицию и стреляет по ситуации.

## Быстрый старт

```bash
python participant_bot_template/main.py 127.0.0.1 9001
```

Запуск браузерного матча из корня репозитория:

```bash
python scripts/start_browser_match.py \
  --bot-a-cmd "{python} participant_bot_template/main.py {host} {port}" \
  --bot-b-cmd "{python} participant_bot_template/main.py {host} {port}"
```


## Браузерный запуск

Матч бот против бота:

```bash
python scripts/start_browser_match.py
```
